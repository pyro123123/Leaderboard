
# Commands Usage

Money Command:
- /money create [Amount] [playerName]
- /money remove [playerName]
- /money get [playerName]

Leaderboard Command:
- /lb create [id] [text]
- /lb remove [id]
- /lb get [id]
- /lb edit [id] [new text]
- /lb list
- /lb load
- /lb money id]
